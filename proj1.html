Airline Analysis

introduction: This project aims to Analyze and visualize Airline data obtained from kaggle.com to uncover trends and insights about the data.
Objective: This project aims to analyze and visualize data using Microsoft Power BI.
Description of the Dataset: The dataset has variables such as Passenger ID, Nationality of passengers, Age, Airports, Flight Status, and Gender.
Prepare Data: Added Age Range using age for better Analyzation.
Data Visualization: Added Cards for Passengers and Airports, Added Slicer for continents, Added ArcGIS Map, Added Stacked Bar Chart for Nationalities, Added Stacked Column Chart with Drill Down for Age-Range and Age, Added Donut and Pie Charts for Gender and Flight Status.

Spotify Analysis

introduction: This project aims to Analyze and visualize Spotify data obtained from kaggle.com to uncover trends and insights about the data.
Objective: This project aims to analyze and visualize data using Microsoft Power BI.
Description of the Dataset: The dataset has variables such as Track, Album Name, Artist, Date, All Time Rank, Track Score, Streams, Playlist Count, and Youtube Views.
Prepare Data: Removed Data such as tiktok views and more, removed duplicate columns, and changed the data type of All Time Rank.
Data Visualization: Added Cards for Songs, Albums, and Artists, Added Stacked Bar Chart for Top 10 Artists, Added Stacked Column Chart for Top 5 Songs by Track Score, Added Donut Chart for Top 5 Songs in Playlists, Added Line Chart for Average Spotify Streams VS Average YouTube Views, Added Area Chart for Spotify Streams by Year.
