Online Sales

Introduction: This project aims to analyze dataset obtained from Kaggle.com, the dataset contains online sales details of a company.
Objective: The objective of this project is to analyze the dataset using Microsoft Excel.
Description of the Dataset: The dataset includes variables such as Transaction ID, Product Category, Units Sold, Total Revenue, Region, and Payment Method.
Data visualization: Added 3 Visualizations such as Line, Stacked Column, and Clustered Bar Charts. Added 3 Slicers for Region, Product Category, and Payment Method.
